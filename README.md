# React Router
<img width="2880" height="1474" alt="image" src="https://github.com/user-attachments/assets/1e8378ff-7919-4cce-8472-d429516265fd" />

A React Router learning project built with React and Vite. This repository contains practical examples of modern client-side routing, including nested routes, dynamic routes, data loading, form actions, protected routes, authentication, and error handling.

## Features

* Client-side routing with React Router
* Nested layouts and routes
* Dynamic route parameters
* Loaders and Actions
* Form handling
* Protected routes
* Authentication flow
* Error boundaries
* Lazy loading
* Modern React Router patterns

## Tech Stack

* React
* React Router
* Vite
* JavaScript (ES6+)
* CSS

## Getting Started

### Clone the repository

```bash
git clone https://github.com/Mllkmoha/react-router.git
cd react-router
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

## Learning Goals

This project explores the core concepts of React Router, including:

* Route configuration
* Nested routing
* Dynamic routing
* Data fetching with loaders
* Mutations with actions
* Navigation and redirects
* Authentication and protected routes
* Error handling
* Route-based code splitting
